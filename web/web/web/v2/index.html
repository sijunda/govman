Simple & Powerful Commands

Everything you need, nothing you don't

bash 
# Install the latest Go version
govman install latest

# Install a specific version
govman install 1.21.5

# Switch Go version globally
govman use 1.21.5 --global

# Set project-specific version
govman use 1.20.1 --local

# List installed versions
govman list

# List all available versions
govman list --all

# Clean up old versions and cache
govman clean --cache

# Update GOVMAN itself
govman self-update

Frequently Asked Questions

Get answers to common questions

Installing GOVMAN is straightforward. You can use the automatic installation script with curl: curl -fsSL https://raw.githubusercontent.com/justjundana/govman/main/scripts/install.sh | bash or with wget: wget -qO- https://raw.githubusercontent.com/justjundana/govman/main/scripts/install.sh | bash After installation, restart your terminal or run source ~/.bashrc (or the appropriate shell config file).

To install a Go version, use govman install 1.21.5 or govman install latest for the newest version. To switch versions, use govman use 1.21.5 for session-only changes, govman use 1.21.5 --default for system-wide changes, or govman use 1.21.5 --local to set a version for your current project.

GOVMAN automatically detects project-specific Go versions using a .govman-version file in your project directory. Create this file with govman use 1.21.5 --local or manually with echo "1.21.5" > .govman-version. When you enter a project directory, GOVMAN will automatically switch to the specified version if shell integration is enabled.

Yes! GOVMAN is designed with security in mind. All downloads are verified with checksum validation and integrity checks. It installs in userspace without requiring admin privileges, doesn't modify system files, and has been tested in production environments. It's safe for both development and production use.

You can update GOVMAN using the built-in self-update command: govman selfupdate. This will download and install the latest version of GOVMAN itself. You can also manually download new releases from the GitHub releases page.

GOVMAN provides tools for efficient disk space management. Use govman clean to remove download cache and temporary files. You can also uninstall specific Go versions with govman uninstall 1.20.0 to free up space. The smart caching system helps optimize storage usage while maintaining performance.

GOVMAN provides native support for Linux, macOS, and Windows operating systems. It integrates with bash, zsh, fish, and PowerShell shells. The smart shell integration automatically manages PATH settings and can be initialized with govman init. This ensures consistent behavior across all supported platforms and shells.

To completely remove GOVMAN, run the uninstall script: curl -fsSL https://raw.githubusercontent.com/justjundana/govman/main/scripts/uninstall.sh | bash This will remove all GOVMAN files, configurations, and installed Go versions. Make sure to backup any important configuration before running this command.
